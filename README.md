ExplorIt is a modern platform designed to help users discover, learn, and explore new ideas effortlessly. Whether you're diving into tech, creativity, or personal growth, ExplorIt brings knowledge to your fingertips with a clean interface and intuitive experience. Our mission is simple: make exploration easy, inspiring, and accessible for everyone.

This project was built using a modern full-stack architecture, focusing on performance, scalability, and clean user experience.

On the frontend, I used Next.js to leverage server-side rendering and optimized performance, along with React for building dynamic and reusable UI components. Styling was implemented using modern CSS techniques to ensure a responsive and visually appealing design across all devices.

For the backend, I developed a RESTful API to handle business logic and data management, deployed on a cloud platform for reliability and scalability. The database is powered by PostgreSQL, providing a robust and efficient data storage solution.

The project is deployed using Vercel for the frontend and Render for the backend, ensuring smooth CI/CD and fast global delivery.

Additionally, I used Git & GitHub for version control, enabling efficient collaboration and continuous development.
